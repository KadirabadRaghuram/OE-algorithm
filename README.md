# 🚀 Optimization Algorithms Dashboard

🔗 **Live App:** https://oe-algorithm-kadirabadraghuram.streamlit.app/

---

## 📌 Overview

This project is an **interactive optimization dashboard** built using Python and Streamlit.
It demonstrates multiple optimization algorithms with **real-time visualization, animations, and user controls**.

The application is designed with an **industry-style UI dashboard**, making it both educational and professional.

---

## 🎯 Features

* 📊 Interactive dashboard with modern UI
* 🎛 Sidebar controls for real-time parameter tuning
* 📈 Live graphs and animated visualizations
* 📉 Performance tracking with metrics and progress bars
* 🌐 Fully deployed web application (no installation required)

---

## 🧠 Algorithms Implemented

### 1️⃣ Unconstrained Minimization

* Gradient Descent method
* Dynamic convergence visualization

### 2️⃣ Pareto Front (Multi-objective Optimization)

* Identifies optimal trade-off solutions
* Scatter plot visualization

### 3️⃣ Genetic Algorithm (Knapsack Problem)

* Population-based optimization
* Selection, crossover, mutation
* Fitness evolution graph

### 4️⃣ Simulated Annealing

* Probabilistic optimization technique
* Temperature-based convergence

## ▶️ How to Run Locally

### 1. Clone the repository

```
git clone https://github.com/KadirabadRaghuram/OE-algorithm.git
cd OE-algorithm
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the app

```
streamlit run Home.py
```
